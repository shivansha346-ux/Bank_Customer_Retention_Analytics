## 📌 Project Overview
This project shifts away from traditional demographic-based churn models to evaluate retail banking retention through behavioral metrics and product depth utilization. Analyzed a global asset portfolio of **10,000 banking customers** to mitigate "silent churn" and optimize retail capital stability.

## 📊 Core Performance Metrics & KPIs
*   **Systemic Attrition (Average Churn Rate):** **20.37%** 
*   **Engagement Retention Ratio (ERR):** **0.53** (Active engagement lowers churn probability by 47%).
*   **Optimal Product Sweet Spot:** Exactly **2 products** yield peak customer stickiness, whereas 3–4 products cause friction and elevate churn.
